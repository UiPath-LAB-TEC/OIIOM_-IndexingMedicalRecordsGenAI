Import AI_MRI_PatientDemoFiles and AI_MRI_Patients Entity from the Data Service Json file. First one is used to store the customer documents and the latter is used for the user specific data storing. 
Upload the three files from the "Sample Documents" folder into the "file" field of the AI_MRI_PatientDemoFiles entity. 
Create App using the app file provided in the "Apps" folder and publish the process in the "Workflow" folder. 

Repair the app by importing the process and the entity, and you're good to go.
